# Admin User Manual

## Login

Go to:

```text
/login
```

Use the admin email and password created during seed.

- Analytics
- Users

- Themes

- Logs

- Permissions

## Reviews

New reviews are created as unapproved. Admin should approve only authentic and appropriate reviews.

## Blog

Blog supports:

- Drafts
- Publishing
- Scheduling data model
- SEO title
- SEO description
- Categories
- Tags
- Featured image

## Leads

Contact and popup leads are saved to the database and emailed to DigiSparkX when SMTP is configured.
