# Component Documentation

## `SiteShell`

Includes:

- Sticky header
- Mega menu
- Search
- Dark/light toggle with saved preference
- Mobile menu
- Footer
- Newsletter area
- Google Maps embed
- 10-minute popup lead form
- Magic click particles and subtle UI sound

## `ContactForm`

Client-side form that posts to `/api/contact`.

## `ReviewForm`

Visitor rating form that posts to `/api/ratings`.

## `ServiceCard`

Shows service category, description, price, features and CTA.

## `CourseCard`

Shows course duration, difficulty, mentor, certificate and enroll CTA.

## `PortfolioCard`

Shows image, description, technology, review, live preview and GitHub link.

## `SectionHeader`

Reusable section heading with eyebrow, title and supporting text.
