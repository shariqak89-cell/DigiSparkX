# Maintenance Guide

## Weekly

- Review leads.
- Approve/reject testimonials.
- Check website forms.
- Review analytics.

## Monthly

- Update dependencies after testing.
- Backup PostgreSQL database.
- Review SEO pages.
- Add new portfolio items and blog posts.
- Check page speed.

## Security

- Rotate admin password periodically.
- Keep SMTP and database secrets private.
- Review admin users and permissions.
- Enable 2FA on hosting and GitHub accounts.

## Content

Use the dashboard and CMS APIs to update:

- Services
- Courses
- Portfolio
- Testimonials
- Blog posts
- Pages
- Media
- SEO settings
